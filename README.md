# Employee management
Proyecto de prueba de una API REST en Spring Boot con Angular.

Tecnologías utilizadas para la contrucción de los Microservicios:
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- Angular 13.3.4
- Bootstrap 4.6
